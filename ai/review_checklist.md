# Review Checklist

This file is the baseline checklist for reviewing AI-generated output before acceptance.

## Review points

- The output matches the requested scope.
- No unsupported assumptions were introduced.
- Naming and structure match local conventions.
- Failure paths and edge cases are handled or documented.
- Security-sensitive behavior is explicit.
- Tests and docs are aligned with the change.

## Reviewer action

Reject any output that changes semantics the repository does not define.
