# AI Guidelines

This directory defines how AI is expected to assist in this project.

It does not contain tooling or automation.
It defines **rules, boundaries, and expectations**.

---

## Purpose

AI is used as a scoped assistant:
- to accelerate reasoning
- to draft code or documentation
- to surface edge cases

AI is not autonomous and does not make final decisions.

---

## Contents

- `prompting_guidelines.md`  
  How to structure prompts with proper context and constraints.

- `coding_rules.md`  
  Mandatory coding standards that apply to all generated code.

- `review_checklist.md`  
  Checklist used to review AI-generated outputs.

- `test_generation.md`  
  Rules for generating tests with AI assistance.

---

## Non-goals

- No vendor-specific instructions
- No automatic code execution
- No implicit trust in AI outputs

All AI-generated work requires human validation.
