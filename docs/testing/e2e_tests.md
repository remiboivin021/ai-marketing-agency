# End-To-End Tests

This document defines the scope and intent of end-to-end validation.

## Cover with E2E

- Critical user journeys
- Boundary integrations
- High-risk regressions with cross-component impact

## Avoid

Do not use end-to-end tests for logic that can be validated faster at a lower level.
