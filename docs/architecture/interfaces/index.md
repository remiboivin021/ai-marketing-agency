# Interfaces

This document is the entry point for interface documentation and explains which files to use for which type of interface.

## Contents

- `api.md` for API interface documentation
- `cli.md` for CLI interface documentation
- `file.md` for file-based interface documentation
- `event.md` for event-based interface documentation

## Maintenance rule

When the system interface inventory changes, update the matching interface document in the same change stream.
