# AI Context

This directory contains the **AI-specific contextual knowledge** of the project.

It defines how AI models, agents, and prompts are configured, constrained, and expected to behave within this project.

---

## Contents

- `model_context.md`
  Model selection, capabilities, limitations, and usage constraints.

- `prompt_context.md`
  Prompt engineering patterns, guidelines, and conventions.

- `agent_context.md`
  Agent configuration, behavior expectations, and orchestration rules.

- `safety_context.md`
  Safety guardrails, content policy, and responsible AI constraints.

---

## Rules

- Keep documents explicit and up to date
- Document actual behavior, not aspirational behavior
- Update when models, providers, or safety requirements change
- Prefer clarity over completeness

If AI context is missing or outdated, fix it before coding.
